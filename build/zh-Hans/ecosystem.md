## How to add a plugin to the list[​](#how-to-add-a-plugin-to-the-list "Direct link to How to add a plugin to the list")

To add a plugin in the community section, these are the steps to do:

1. Start contributing to the Fastify project following the [contributing guide](https://github.com/fastify/fastify/blob/main/CONTRIBUTING.md) and read the [on boarding process](https://github.com/fastify/.github/blob/main/GOVERNANCE.md#collaborator-nominations).

2. Fork the [Fastify repository](https://github.com/fastify/fastify) on GitHub

3. Add the plugin in the community list in `docs/Guides/Ecosystem.md` file.

4. Submit the PR!

As soon as the PR is reviewed and merged, the plugin will appear in this page!
